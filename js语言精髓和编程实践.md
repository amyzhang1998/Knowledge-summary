## 语法作用域

1. if语句中的大括号被用作标识’复合语句‘
2. 把复合语句替换成单行，报错，大括号就是语法结构的一部分。
语法作用域的级别
1. 相同级别的语法作用域可以相互嵌套。
2. 高级别的语法作用域能够包含低级别的语法作用域，
3. 低级别的不能包含高级别的

```
//1
if(true){
    //...
}
function foo(){
//...
}
//2
if(true){
    //...
    function foo(){
        //...
    }
}
1和2完全等效，因为语句无法包含比他等级更高的函数语法作用域
```
